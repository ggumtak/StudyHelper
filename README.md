# Study Helper

> Code/quiz practice workspace powered by a local server.

## Getting Started
1. Run `StudyHelper.exe`.
2. Your browser opens automatically.
3. Pick a file/mode and start practicing.

## Folder Layout
```
StudyHelper/
├── StudyHelper.exe    # run this
├── ngrok.exe          # optional for external access
├── data/              # practice sources
├── config/            # config files
└── src/               # source code
```

## Practice Modes
| Mode | Description |
|------|-------------|
| 1 | C# OOP fill-in-the-blanks |
| 2 | Data structure blanks |
| 3 | Blank sheet coding |
| 4 | Mock exam |
| 5 | Definition quiz |
| 6 | Computational math coding |
| 7 | Vocabulary |

## Shortcuts
- `Enter` - next blank
- `Ctrl+Enter` - grade all
- `Shift+Enter` - grade current (Mode 3)
- `S` - shuffle questions
- `R` - reset

## Mobile Access
After the server starts on PC, open the shown address from a phone on the same Wi-Fi.
