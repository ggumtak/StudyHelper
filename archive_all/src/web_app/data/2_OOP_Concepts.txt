메서드 오버로딩 (Method Overloading),같은 이름의 메서드를 매개변수만 다르게 여러 개 정의하는 것
인터페이스 (Interface),구현해야 할 메서드 규약만 정의한 틀로 클래스가 이를 구현해 사용함
다중 상속 (Multiple Inheritance),한 클래스가 둘 이상의 부모로부터 상속받는 개념(※C#은 클래스 다중 상속은 불가, 인터페이스는 가능)
정적 멤버 (Static Member),객체 생성 없이 클래스 자체에서 직접 사용하는 변수나 메서드
구조체 (Structure),값 타입으로 작동하는 간단한 데이터 묶음(작은 데이터 저장에 적합)
속성 (Property),필드를 안전하게 다루기 위한 get/set 접근자를 가진 멤버
델리게이트 (Delegate),메서드를 변수처럼 참조해 호출할 수 있는 타입(콜백 구현에 사용)
식 람다 (Expression Lambda),간단한 람다식을 '=>' 형태로 표현한 익명 함수
예외 처리 (Exception Handling),프로그램 오류를 try-catch로 안전하게 처리하는 구조
쓰레드 (Thread),프로그램 내에서 동시에 실행되는 작업 흐름 단위
