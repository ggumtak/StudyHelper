메서드 오버로딩(Method Overloading)
같은 이름의 메서드를 매개변수만 다르게 여러 개 정의하는 것

인터페이스(Interface)
구현해야 할 메서드 규약만 정의한 추상 타입

다형성(Polymorphism)
같은 타입이지만 실행 결과가 다양한 객체를 대입할 수 있는 성질

캡슐화(Encapsulation)
객체의 데이터를 외부에서 직접 접근하지 못하게 하고 메서드를 통해서만 접근하도록 하는 것

상속(Inheritance)
부모 클래스의 속성과 메서드를 자식 클래스가 물려받는 것

추상 클래스(Abstract Class)
구현되지 않은 추상 메서드를 포함할 수 있는 클래스로 직접 인스턴스화할 수 없음

생성자(Constructor)
객체가 생성될 때 자동으로 호출되어 객체를 초기화하는 특수한 메서드

정적 멤버(Static Member)
클래스 수준에서 공유되며 인스턴스 없이 클래스명으로 직접 접근할 수 있는 멤버

델리게이트(Delegate)
메서드를 변수처럼 참조하여 호출할 수 있는 타입

예외 처리(Exception Handling)
프로그램 오류를 try-catch로 안전하게 처리하는 구조
